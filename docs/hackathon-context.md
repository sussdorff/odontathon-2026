### Challenge 2: "Agent nach dem Stuhl" — Practice Orchestrator

## Key Insights from Setup Call

1. **Speed = Conversion** — longer patient consideration time = more rejections
2. **Documentation gaps = lost revenue** — incomplete docs → missing billing positions
3. **Sonja (competitor) underperforms** — high error rate, no learning after 4-5 months
4. **Local agent = §203 advantage** — no cloud data processing issues
5. **"Agent im Ohr" vision** had strongest emotional momentum

## Our Competitive Advantages

- Rules as Data — billing rules configurable, not hardcoded
- Local-first — §203-compliant, no cloud dependency
